# laravel-nova-many-to-one-polymorphic-relationship
Drop-in package that adds a many-to-one (inverse of one-to-many) polymorphic relationship to Eloquent and Nova.

## Version Support

| PHP   | Laravel | Nova | Package |
|-------|---------|------|---------|
| 8.2   | 10, 11  | 4.x  | 1.x     |
| 8.3   | 10, 11  | 4.x  | 1.x     |
| 8.4   | 10, 11  | 4.x  | 1.x     |
| 8.5   | 10, 11  | 4.x  | 1.x     |
